IPM RAS PSK and FS spectrometer files viewer

Required packages:
  • et-xmlfile
  • numpy
  • openpyxl
  • pandas
  • pyqtgraph
  • PySide6
  • python-dateutil
  • pytz
  • scipy
  • shiboken6
  • six


Required Python: >= 3.8

For developers. To add a translation, use
   pylupdate5 -noobsolete *.py -ts xy_XY.ts
   lrelease *.ts
